# OPPO ads update notes

## Version

1.0.16+17

## Store declaration

- Contains ads: Yes
- Ad format: Banner ad
- Ad network: Google AdMob
- Internet use: Required for loading ads only. The debt ledger features remain local/offline.

## Current build mode

This release includes a real Google AdMob banner at the top of the main screens. If no ad is available, the ad area is hidden.

Declare ads in the store listing before submitting this build.
