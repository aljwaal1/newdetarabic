# دفتر الديون - fix8 safe test ads

تطبيق Flutter عربي يعمل أوفلاين لتسجيل الديون بين الأشخاص.

## ما الجديد في fix8

- تشغيل التطبيق أولًا ثم محاولة تحميل الإعلان بعد ظهور الواجهة.
- إخفاء مساحة الإعلان تلقائيًا إذا فشل AdMob على جهاز قديم أو جهاز Huawei بدون خدمات Google كاملة.
- استخدام إعلانات Google الاختبارية فقط لمعاينة شكل الإعلان.
- الرجوع إلى نسخة AdMob أكثر توافقًا مع الأجهزة القديمة.
- تحديث رقم النسخة إلى `1.0.8+9`.
- تحديث اسم Artifact في GitHub Actions إلى:
  `debt-pal-arabic-v1-fix8-safe-test-ads-apk`

## ملاحظة مهمة

هذه النسخة مخصصة للتجربة فقط لأنها تستخدم إعلانات اختبارية. لا ترفعها كنسخة ربح نهائية على المتجر.

بعد التأكد من أن شكل الإعلان مناسب وأن التطبيق يفتح على Huawei، يمكن بناء نسخة أخرى بأرقام AdMob الحقيقية.

## البناء عبر GitHub Actions

ارفع الملفات إلى GitHub ثم شغّل workflow من:

`.github/workflows/build-apk.yml`

سيتم إنشاء APK داخل Artifact باسم:

`debt-pal-arabic-v1-fix8-safe-test-ads-apk`
