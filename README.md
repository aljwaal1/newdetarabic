# دفتر الديون - Debt Pal Arabic V1

تطبيق ديون عربي أوفلاين بتصميم داكن ناعم مستوحى من الصور التي أرسلتها.

## المزايا
- واجهة داكنة ناعمة.
- زر + كبير في الأسفل.
- قائمة دائرية: أعطيت / أخذت / إغلاق.
- إدخال خطوة خطوة: اسم الشخص ثم المبلغ.
- لوحة أرقام كبيرة.
- ملاحظات وموعد سداد سريع.
- الصفحة الرئيسية تعرض: لي عند الناس، علي للناس، الصافي.
- نفس الشخص يظهر مرة واحدة فقط في الرئيسية.
- صفحة شخص بتايم لاين للعمليات.
- سحب العملية يمينًا للتسوية ويسارًا للحذف.
- إعدادات بأيقونات دائرية.
- العملات: دولار، دينار، ريال، جنيه، درهم.
- الافتراضي: دولار.
- حفظ محلي داخل الهاتف.
