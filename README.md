# دفتر الديون - fix6 share text

تطبيق Flutter عربي يعمل أوفلاين لتسجيل الديون بين الأشخاص.

## ما الجديد في fix6

- إصلاح خطأ مهم في `lib/main.dart`: كان يوجد تعريف مكرر للدالة `appFeedback`، وتم الإبقاء على النسخة الصحيحة التي تدعم الصوت والاهتزاز.
- تحسين مشاركة كشف الحساب كنص كامل من صفحة الشخص.
- كشف الحساب النصي أصبح يشمل:
  - اسم الشخص.
  - تاريخ الكشف.
  - إجمالي "لي عنده".
  - إجمالي "علي له".
  - الصافي.
  - كل العمليات الخاصة بالشخص، حتى العمليات المسددة.
  - المبلغ الأصلي، المسدد، المتبقي، الحالة، تاريخ العملية، موعد السداد إن وجد، والملاحظة إن وجدت.
- تحديث رقم النسخة إلى `1.0.6+7`.
- تحديث اسم Artifact في GitHub Actions إلى:
  `debt-pal-arabic-v1-fix6-share-text-apk`

`.github/workflows/build-apk.yml`
